### Print Specifications

| Element | Specification | Why |
|---------|--------------|-----|
| **Line weight** | 2-3pt borders | Thin lines vanish on inkjet printers |
| **Font size** | 16pt minimum (problems), 20pt+ for Grades 1-2 | Readability for small hands |
| **Margins** | 0.75" all sides | Home printers clip edges |
| **Colors** | Black lines on white only | No gray fills that waste ink |
| **Grid cells** | Minimum 1" Ã— 1" | Room to write and color |
| **Page count** | Single page per puzzle | Flipping pages breaks focus |

### Printing Tips

1. **Select "Fit to Page"** in print settings
2. **Choose "Black & White"** mode to save ink
3. **Card stock** (65-80 lb) prevents marker bleed-through
4. **Crayons or colored pencils** work better than markers
