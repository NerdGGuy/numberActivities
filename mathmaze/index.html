How to Play Math Maze

1

Look at the Target Color box - it shows which color to find!

2

The entire path is ONE color. Find cells where the math equals that number.

3

Solve math problems. If the answer equals the target number, it's on the path!

4

Select the target color and click path cells to color them.

5

Find all path cells from START to END to win!

Tips:

  • ALL path cells have the same answer - the target number!
  • The path only goes right or down.
  • Decoy cells have different answers - don't be fooled!
  • Use the eraser to fix mistakes.
  • Watch your progress bar to see how many path cells you've found!
