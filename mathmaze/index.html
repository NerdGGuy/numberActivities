How to Play Math Maze

1

Look at the Color Key to see which answers match which colors.

2

Find the START cell (green) and the END cell (red).

3

Solve each math problem in the maze cells.

4

Select a color from the palette, then click the cell to color it.

5

Color all the path cells correctly to reveal the solution!

Tips:

  • The correct path only goes right or down.
  • Use the eraser to fix mistakes.
  • Click Check Answers to see how you're doing!
